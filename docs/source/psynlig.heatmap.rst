psynlig.heatmap
---------------

The :ref:`psynlig.heatmap <api-heatmap-source>` module defines
methods for generaing a heat map to show
`Pearson correlation coefficients <https://en.wikipedia.org/wiki/Pearson_correlation_coefficient>`_ for pairs of variables.
